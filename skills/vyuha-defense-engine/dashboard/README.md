# Dashboard Visualizations (Ideas)

Each Vyuha formation should render as:
- a **mode card** (active/inactive)
- an **explainability panel** (triggers + reasons + confidence)
- a **control plane panel** (current defensive controls applied)
- an **evidence panel** (what is being collected and why)
- a **recovery panel** (how to unwind safely)

Recommended visual metaphors:
- Chakra: concentric rings (outer perimeter controls + inner protected core)
- Padma: petals (segmented zones with different controls)
- Vajra: strong central spine with mirrored control sets
- Mandala: multi-ring governance with strict inner sanctum
