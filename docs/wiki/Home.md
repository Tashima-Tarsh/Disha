# DISHA OS Wiki

DISHA OS is an **AI Cyber Evidence Operating System** for secure automation, monitoring, and evidence-grade auditability.

Key principles:
- Local-first by default (runs as a single-machine appliance)
- Zero-trust in every layer (auth + explicit policy checks)
- Evidence-grade logging (audit trails, reasons, and request IDs)
- Deterministic behavior for critical flows (timeouts + bounded outputs)
- Graceful degradation (no single dependency takes the system down)
