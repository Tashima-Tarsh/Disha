# Technical Stack

DISHA utilizes a curated selection of enterprise-grade technologies to deliver high-performance autonomous intelligence.

## ⚡ Core Runtimes

| Component | Technology | Rationale |
|:--- |:--- |:--- |
| **Primary Runtime** | [Bun](https://bun.sh/) | Ultra-fast JS/TS execution and native workspace management. |
| **Intelligence Layer** | [Python 3.13](https://python.org) | Unmatched ecosystem for ML (PyTorch, Scikit-Learn) and Scientific Computing. |
| **Concurrency** | [FastAPI](https://fastapi.tiangolo.com) | High-performance, async-native backend for AI microservices. |

---

## 🧠 Intelligence & AI

- **Reasoning**: Custom 7-stage cognitive loop implementing multi-agent consensus.
- **Models**: Unified SDK supporting Claude 3.5, GPT-4o, and local LLaMA/Ollama deployments.
- **Machine Learning**:
  - **Graph AI**: PyTorch Geometric for link prediction and knowledge graph analysis.
  - **RL**: Custom PPO (Proximal Policy Optimization) for strategic resource management.
  - **Physics**: MDEngine (Molecular Dynamics) for structural simulations.

---

## 📦 Data & Persistence

- **Vector Memory**: [ChromaDB](https://trychroma.com) for semantic similarity search and RAG.
- **Graph Memory**: [Neo4j](https://neo4j.com) for entity-relationship mapping and sovereign graph intelligence.
- **Message Bus**: [Apache Kafka](https://kafka.apache.org) for high-frequency OSINT and telemetry streams.
- **Caching**: Local filesystem and Redis-compatible in-memory caches.

---

## 🎨 User Experience

- **Framework**: [Next.js 16](https://nextjs.org) (App Router).
- **UI Components**: High-end React components with custom HSL design tokens.
- **Styling**: Vanilla CSS + Tailwind CSS for luxury typography and immersive glassmorphism.
- **Client Runtime**: Bun-native bundling for ultra-fast load times.

---

## 🛠️ DevTools & Security

- **Linter/Formatter**: [Biome](https://biomejs.dev) (TS/JS) and [Flake8/Black](https://flake8.pycqa.org) (Python).
- **Bundler**: [esbuild](https://esbuild.github.io) for CLI distribution.
- **Observability**: [OpenTelemetry](https://opentelemetry.io) for enterprise-tier tracing and logs.
- **Security**: CodeQL for SAST and custom Sentinel shields for runtime DAST.
