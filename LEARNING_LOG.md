### рҹ©№ Self-Healing Mechanisms

1. **Checkpoint gating** вҖ” New models only promoted if metrics improve (5% regression tolerance)
2. **Hyperparameter auto-tuning** вҖ” Stagnation detection bumps learning rate; high loss triggers regularization
3. **Fallback to synthetic data** вҖ” If network fetch fails, training continues with generated scenarios
4. **Safe rollback** вҖ” Previous checkpoints preserved; staging directory cleaned after promotion
5. **Cross-domain validation** вҖ” Knowledge graph validates that all 8 domains contribute to training

### рҹ”Ғ How to Run Continuous Learning

```bash
# Full pipeline (all components, online data)
python scripts/continuous_train.py --rounds 3

# Offline mode (synthetic data only)
python scripts/continuous_train.py --rounds 3 --offline

# Single component
python scripts/continuous_train.py --rounds 5 --component rl
python scripts/continuous_train.py --rounds 5 --component gnn
python scripts/continuous_train.py --rounds 5 --component decision
python scripts/continuous_train.py --rounds 5 --component knowledge

# Train all (single pass)
python scripts/train_all.py
```
