### 📊 Reinforcement Learning Engine

| Component | Detail |
|-----------|--------|
| **Algorithm** | Proximal Policy Optimization (PPO) |
| **State** | 12-dim (entities, anomalies, risk, depth, agents_used, time) |
| **Actions** | 8 (run 5 agents, increase/decrease depth, stop) |
| **Reward** | Entity discovery (+0.1), anomaly (+0.5), risk change (×2.0), time penalty (-0.01) |
| **Policy** | Actor-Critic MLP (12→64→64→8) with action masking |
| **Replay** | Prioritized experience buffer (10K transitions, α=0.6) |
| **Training** | 400 episodes, average reward 22.24 (±3.23) |

| **Link Prediction** | Cross-domain entity relationships |
| **Knowledge Graph** | ~99.8% accuracy on real cross-domain graph |
| **Features** | Early stopping, shuffled splits, weight decay (5e-4) |

```bash
# Clone the repository
git clone https://github.com/Tashima-Tarsh/Disha.git
cd Disha

# Install CLI dependencies
bun install

# Build the CLI
bun run build

# Type check
bun run typecheck

# Lint
bun run lint
```

```bash
# 💻 CLI Engine
bun src/entrypoints/cli.tsx

cd cyber-defense && docker-compose up -d

# 🌐 Web Dashboard (port 3000)
cd web && npm install && npm run dev
```
