# Integrations

This directory contains imported external repositories integrated into the Disha platform.

## Repositories

### 1. [Anthropic](./Anthropic/)

- **Source:** [Tashima-Tarsh/Anthropic](https://github.com/Tashima-Tarsh/Anthropic) (branch: `ThariqS-patch-1`)
- **Description:** Claude Code fork with plugins, commands, and devcontainer support.
- **Stack:** TypeScript, Claude CLI

### 2. [osint-analyser](./osint-analyser/)

- **Source:** [Tashima-Tarsh/osint-analyser](https://github.com/Tashima-Tarsh/osint-analyser) (branch: `main`)
- **Description:** Python OSINT tool for intelligence gathering via Telegram, LLM analysis, and MySQL storage.
- **Stack:** Python, Telegram API, LLM, MySQL, Docker

### 3. [Pentagi](./Pentagi/)

- **Source:** [Tashima-Tarsh/Pentagi](https://github.com/Tashima-Tarsh/Pentagi) (branch: `main`)
- **Description:** AI-powered Penetration Testing platform with automated security assessment capabilities.
- **Stack:** Go, React, Docker, GraphQL

### 4. [cyber-intelligence-platform](./cyber-intelligence-platform/)

- **Source:** [Tashima-Tarsh/cyber-intelligence-platform](https://github.com/Tashima-Tarsh/cyber-intelligence-platform) (branch: `main`)
- **Description:** Python cyber intelligence platform for OSINT aggregation, threat classification, alerting, and automated reporting.
- **Stack:** Python

### 5. [Tash](./Tash/)

- **Source:** [Tashima-Tarsh/Tash](https://github.com/Tashima-Tarsh/Tash) (branch: `main`)
- **Description:** AI Public Auditor platform for production-grade auditing with GraphQL API and Next.js frontend.
- **Stack:** Go, Next.js, GraphQL, TypeScript, Docker
- **Structure:** `backend/`, `frontend/`, `docs/`, `infra/`, `README.md`

## Notes

- Binary files (images, fonts, compiled binaries, etc.) were skipped during import. Placeholder `.SKIPPED.md` files are left in their place with metadata about the original file.
- `__pycache__` directories were excluded from the import.
