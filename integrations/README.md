# Disha Integrations

This directory contains code and files imported from external repositories that form part of the Disha unified platform.

| Repository | Directory | Description | Source |
|---|---|---|---|
| Anthropic (Claude Code fork) | `Anthropic/` | Claude Code with plugin architecture | [GitHub](https://github.com/Tashima-Tarsh/Anthropic) |
| OSINT Analyser | `osint-analyser/` | Telegram OSINT collection + LLM analysis | [GitHub](https://github.com/Tashima-Tarsh/osint-analyser) |
| Pentagi | `Pentagi/` | AI Penetration Testing (Go + React) | [GitHub](https://github.com/Tashima-Tarsh/Pentagi) |
| Cyber Intelligence Platform | `cyber-intelligence-platform/` | Cyber intelligence & threat monitoring | [GitHub](https://github.com/Tashima-Tarsh/cyber-intelligence-platform) |
| Tash | `Tash/` | AI Public Auditor (Go + Next.js + GraphQL) | [GitHub](https://github.com/Tashima-Tarsh/Tash) |

## Purpose

These integrations extend Disha's capabilities with:
- **OSINT collection** from open sources via Telegram bots
- **AI-powered penetration testing** with 20+ security tools
- **Cyber intelligence** gathering and threat analysis
- **Public AI auditing** for transparency and accountability
- **Enhanced Claude Code** with plugin support
