# Cyber Intelligence Platform

A Python-based cyber intelligence platform for threat monitoring, OSINT aggregation, and automated reporting.

## Structure

- `ai/` - AI-based crime classification
- `alerts/` - Alert engine for generating security alerts
- `automation/` - Pipeline automation and git sync
- `connectors/` - Data connectors (NCRB scraper)
- `osint/` - OSINT data aggregation
- `reports/` - Report generation (DOCX)
- `ui/` - FastAPI-based dashboard with charts and maps
- `main.py` - Entry point
